Desarrollador Python Remoto

Kastech Colombia - Bogotá, D.C

Desarrollador Sr. Python - Nuestro equipo de ingeniería es un grupo muy unido de personas trabajadoras y de rápido aprendizaje que cree en nuestra misión de ayudar a las comunidades a prosperar. Creemos en un enfoque basado en datos para la toma de decisiones y en una cultura alentadora que enfatiza el aprendizaje de nuestros errores. Debatimos y deliberamos sobre las mejores ideas para mejorar nuestra pila tecnológica, y nos esforzamos por crear una atmósfera inclusiva que promueva el bienestar psicológico de todos nuestros compañeros de equipo.

Requerimientos

AWS, Java 8 / Dropwizard / aplicaciones de 12 factores, Python 3 / Serverless AWS Lambdas, MySQL & DynamoDB, S3 & Athena,

Jenkins / Github / Terraform.

Tener experiencia previa trabajando en trabajos y servicios de back-end en Java.

Tener fuertes habilidades de análisis, organización y priorización.

Demostrar la capacidad de llevar el código desde los garabatos de la pizarra hasta su ejecución en producción.

Habilidades imprescindibles

Python + AWS

Habilidades que se deben tener

Conocimiento de Terraform".

Responsabilidades

Trabajar en un equipo de hasta 5-6 ingenieros + un gerente de producto en nuestros proyectos más grandes y, a veces, individualmente en mejoras más pequeñas y de tamaño pequeño.

Entregar nuevas características del producto en apoyo de nuestras aplicaciones móviles para los usuarios y paneles de control web para los comerciantes y mejorar la capacidad de nuestros productos existentes para escalar con un usuario creciente y la demanda de los comerciantes.

Construir servicios Java REST para actuar en conjunto con nuestra arquitectura orientada a mensajes basada en AWS y también construir lógica de "pegamento" Python, implementada en AWS Lambda utilizando el marco Serverless.

Requerimientos

  • Idiomas: Inglés
  • Edad: A partir de 18 años
